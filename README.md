# bookstore-inventory-system-C
A beginner-friendly inventory management system in C that allows users to add books, process sales, and generate low-stock reports through a menu-driven interface.


## Overview

This project is a simple command-line inventory management system developed in C. It helps manage bookstore inventory by allowing users to add books, process sales, and generate low-stock reports.

The project was built to practice core programming concepts such as arrays, functions, searching, updating records, and menu-driven applications.

## Features

* Add new books to inventory
* Prevent duplicate ISBN entries
* Process book sales
* Update stock quantities automatically
* Generate low-stock reports
* Menu-driven user interface
* Input validation for inventory capacity

## Technologies Used

* C Programming Language
* Standard C Libraries (`stdio.h`)

## Concepts Practiced

* Functions
* Arrays
* Two-Dimensional Character Arrays
* Searching Algorithms
* Menu-Driven Programming
* Inventory Management Logic
* Data Validation

## How It Works

1. Add books using ISBN, title, price, and quantity.
2. Process sales by entering the book ISBN and quantity sold.
3. Automatically update available stock.
4. Generate a report showing books with fewer than 5 copies remaining.
5. Exit the application when finished.

## Sample Menu

1. Add New Book
2. Process a Sale
3. Generate Low-Stock Report
4. Exit
